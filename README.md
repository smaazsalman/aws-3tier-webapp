# AWS Three-Tier Web Application

## Overview
This project demonstrates deployment of a scalable three-tier web application on AWS.

## Architecture
- EC2 (Application Layer)
- Application Load Balancer
- RDS (Database Layer)
- VPC with public and private subnets

## Features
- Secure networking using security groups
- IAM roles for access control
- High availability setup

## Steps
1. Created VPC and subnets
2. Launched EC2 instances
3. Configured Load Balancer
4. Connected RDS database

## Outcome
Successfully deployed a secure and scalable AWS-based application.
