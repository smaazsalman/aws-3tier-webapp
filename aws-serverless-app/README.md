# AWS Serverless Application

## Overview
Built a serverless application using AWS Lambda, API Gateway, and DynamoDB.

## Architecture
- API Gateway (entry point)
- AWS Lambda (backend logic)
- DynamoDB (data storage)

## Features
- API-based request handling
- Serverless backend processing
- Scalable and cost-efficient design

## Steps
1. Created Lambda function
2. Configured API Gateway endpoint
3. Connected DynamoDB table
4. Tested API requests

## Outcome
Successfully built a scalable serverless backend system on AWS.

## Skills Demonstrated
- AWS Lambda
- API Gateway
- DynamoDB
- Serverless architecture
